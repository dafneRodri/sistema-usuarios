# 🚀 Sistema de Gestión de Usuarios (CRUD)
### Proyecto de Infraestructura de Servidores y Desarrollo Colaborativo

-----------------------------------------------

## 🎯 Objetivo
Implementar una infraestructura de red cliente-servidor utilizando sistemas operativos basados en el núcleo Linux (CentOS y Fedora), con el fin de desplegar una aplicación web funcional. El enfoque principal es el trabajo colaborativo mediante Git/GitHub, la administración de servicios web (Apache), bases de datos (MariaDB) y la seguridad de red (Firewall/SSH).

## 💻 ¿Qué estamos creando?
Estamos construyendo un Sistema de Control de Usuarios que permite realizar las cuatro operaciones básicas de gestión de datos en una base de datos centralizada:
* Registro de usuarios con diseño moderno (Dark Mode).
* Autenticación y Gestión de perfiles.
* Sincronización automática entre desarrolladores y servidor mediante Git.

-------------------------------------------------

## 👥 Equipo de Trabajo y Roles
* Dafne (Server Admin): Instalación de servicios (Apache, PHP, Git), gestión de repositorios y despliegue final en CentOS.
* Darinka (DB & Security): Configuración de MariaDB, gestión de privilegios y reglas de Firewall.
* Patricio (Backend): Desarrollo de la lógica en PHP, scripts de conexión y manejo de datos.
* Abril (Frontend): Diseño de la interfaz de usuario con CSS personalizado y estructura HTML.

-------------------------------------------------

## 🛠️ Tecnologías Utilizadas
* **Servidor:** CentOS Stream
* **Clientes de Desarrollo:** Fedora Workstation
* **Servidor Web:** Apache (httpd)
* **Base de Datos:** MariaDB (SQL)
* **Control de Versiones:** Git & GitHub
